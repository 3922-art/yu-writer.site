Yu Writer

简洁流畅的文本编辑器

为专业技术人员、博客作者以及热爱写作的你而打造
Yu Writer 是一款支持 Markdown 格式的文本编辑器， 支持图文混排、表格、图表、数学公式， 可用于编辑博客文章、技术文档、办公及书籍等电子文档。Yu Writer 通过许多设计改善写作体验，致力为你带来写作的乐趣。

文档库

Yu Editor 使用内置的文档库集中统一地管理文档，以后再也不需要手忙脚乱到处寻找待编辑的文档了。

  • 自动保存 文档会自动保存，你可以随时关闭应用，然后随时打开并继续编辑之前的文档。
  • 历史记录 文档的每次修改都有历史记录，你可以随时对比查看、回滚到之前的任一版本。
  • 多设备同步 文档库可以建立在 Dropbox、OneDrive、Google Drive、Git 等文件夹之上，通过它们把文档同步到你的所有设备。

图片与素材

  • 拖拽插入图片 只需把照片拖入文档，应用就会自动把照片缩放到合适的大小并插入到文档， 同时还会自动生成一份适合 Retina（HiDPI）屏幕观看的版本。
  • 文档媒体库 每个文档均有独立的媒体库，在媒体库里可以集中地管理文档附带的图片、视频、音频等素材。
  • 导出 导出文档时可以连同素材一并打包。

实时大纲及实时预览

  • 实时文档大纲 通过大纲可以方便地把握整篇文档的结构，并且可以快速定位到你所需要的文档位置。
  • 大纲可编辑 可以移动大纲节点以调节段落的顺序及层次关系，在大纲上的修改会同时作用到文档。
  • 实时预览 Markdown 语法生成的格式和排版结果一目了然。
  • 预览缩放比率 可以调整预览的宽度，以不同的缩放比率查看预览。
  • 预览定位 可以拖动预览上面的浮块来快速定位文档位置。
  • 图文混排
  • 可交互的表格
  • 图表
  • 题注的自动编号
  • 自动目录
  • LaTeX 数学公式

格式支持

Yu Writer 支持大部分 Markdown语法，除此之外还支持更多格式以满足更多类型文档的需求，例如：

  • 图文混排 图片可以设置居左、居右、居中等对齐方式，可以设置大中小 3 中不同大小。
  • 图表 支持办公文档常见的饼型图和柱状图等图表。
  • 自动目录（TOC） 支持自动生成文档的目录。
  • 自动编号 支持自动生成图片、表格、图表的题注当中的编号。

模版

虽然使用 Markdown 不能随意改变文字或段落的字体、大小、颜色等格式。 但通过模版可以生成符合不同类型文档所需要的样式。

没有调整文字和段落格式的繁杂工具，还可以让你更专注于书写内容而不是频繁地调整格式。

Yu Writer 内置了涵盖博客文章、技术文档、办公、电子书籍出版物、学术文档等模版， 可以生成整洁而专业的文档。

主题及录入模式

  • 界面主题 Yu Writer 提供几种界面主题，以营造适合你书写时喜欢的环境。
  • 录入模式 另外还提供几种录入模式以从细微之处改善写作体验，比如 Nature 模式可以让输入光标停留在 屏幕中间偏下的地方，而不是屏幕底端。Thinking 模式则会把除了正在录入的几行文字以外的 内容颜色变淡。
  • 可定制界面 界面的每个部分都是可以设置为隐藏的，比如工具栏以及实时预览，以符合你的书写习惯。
  • Statistics
  • Power Mode
  • Typewriter
  • Focus

插件

通过插件可以扩充应用的功能，应用内置了几个插件，比如可以统计编辑时间、录入时间及字数的 Statistics，为文字录入增添视觉和声音效果的 Power Mode 和 Typewriter 等。

For Writers

为重度写作用户特别准备的功能

  • 格式自动转换

    从网页复制过来的内容将会自动转换为 Markdown 文本，无需先粘贴为纯文本或者 HTML 文本再手动修改， 方便资料采集和编写。

  • 并行修改

    可以在两个或多个应用窗口中编辑同一份文档，在不同窗口中编辑文档的不同部分， 所做的修改均会即时同步到其他窗口， 并且每个应用窗口的编辑操作都是相互独立的，配合屏幕分屏或者多显示器，可以很方便地编辑长篇的文档。

  • 一键发布

    写好的文章准备发布了？现在只需一次点击就可以自动发布到 Wordpress， 连同照片上传等操作，应用自动帮你完成。
