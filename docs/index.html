多款主题、可定制的界面

不喜欢繁杂的界面？Yu Writer 窗口可以精简到只保留一个编辑框，在简洁的界面之下，蕴藏着丰富的功能。

所有文档都在一个地方

你的所有创作都在内置的文档库里，再也不用到处寻找文档然后再打开编辑。支持全文搜索、支持 Tag（标签）管理。同时可作为系统默认的 Markdown 和纯文本文件编辑器使用。另外文档库可以建立在 iCloud Drive, Google Drive, OneDrive 和 Dropbox 等云储存的本地文件夹里面，方便在多台设备间同步文档。

文档热保存、自动还原工作现场

你可以随时打开应用开始写作，也可以随时关闭应用，文档是热保存的，再也不会因为忘记手动保存、误操作或者程序奔溃而丢失数据。每次打开应用时还能自动打开上次编辑时的文档以及还原光标位置，方便继续书写。

日志式版本记录

文档的编辑全过程（包括每个文字的录入和删除）都可以记录下来，真正可以做到回滚到历史上任意版本、任意时刻。还支持重播观看创作过程哦！

导出高质量的 PDF 文档以及多种博客文章格式

可以把文档导出为 PDF、WordPress 博客的 HTML、Text Bundle、图片等格式，还支持导出为微信公众号文章格式。导出 PDF 时更可选择不同的样式以将文档渲染成不同的风格。

内置静态站点和博客生成引擎，轻松创作、方便管理

服务端无需安装和配置繁杂的博客程序和数据库软件就可以创建你的站点或者博客。创建、编辑、发布文章都在本应用程序内轻松方便完成，编辑好你的文档，剩余的事情都可以交给 Yu Writer 自动完成，管理本地的文档就是管理你的站点和博客。

内置电子书生成引擎，让你专注内容创作

可生成高质量的 EPUB 电子书，或者适合在线阅读的在线版本。

支持 GitHub Pages、GitBook、Jekyll、Hexo、Hugo 等

可以使用自己喜欢的平台、工具来管理自己的站点和博客、制作电子书。

可编辑的大纲视图，轻松组织超长篇幅的文档

段落的定位、顺序调整、层次结构调整、删除，都能在大纲视图里完成，同时支持大纲搜索。

多标签、多窗口并行编辑，轻松应对重度写作任务

多标签页可轻松切换多个文档，多窗口并行编辑可方便文档内的参照和引用。每个文档的编辑会实时同步到其他窗口而不用担心相互覆盖问题。

文档间的链接跳转

无论在编辑状态下还是阅读模式，均支持文档间和段落间的链接跳转，编辑或阅读多章节的文档、博客或者电子书将会变得轻松简单。
