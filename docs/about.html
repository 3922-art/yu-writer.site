一款能找到写作乐趣的 Markdown 文本编辑器

ABOUT

大约一两年前（指 2015 年），当时我想建一个博客用来分享一些文章，然后发现 现有的博客程序编辑文章的体验都不是很好，特别是当文章篇幅较长、含有较多图片时，偶尔 文字格式会混乱，偶尔会产生一些莫名其妙的代码，有时很难迅速地找到想要的图片。 加上因为是在浏览器当中编辑，滚动条有时也不是按照自己的想法来滚动……

于是我就开始尝试写自己的文本编辑器了。

从 HTML5、ES6 开始学起，到 SASS 到 Node.js 到 NPM 再到 Electron， 边学边写，所以陆陆续续 Yu Writer 写了近一年多。

现在 Yu Writer 的雏形基本上完成了，开始提供测试版供其他同样喜欢写文章的朋友试用，我自己也用它 来写文章，边使用边改进完善。

如果你在使用 Yu Writer 过程中遇到问题或者想提意见，欢迎直接联系我，也欢迎把 Yu Writer 推荐给您的朋友试用，谢谢！
